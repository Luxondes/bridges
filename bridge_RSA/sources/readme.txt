Main installation:
Require a Windows, RSA is a DLL and not version is available for Linux or Mac.
Install Python 64bit and RSA which are contained on the "toinstall" folder.

For launch the script directly, open a shell then write:
-> python fast_RSA.py

If the RSA has be installed on a another harddrive than C:, add the harddrive like follow:
Example: ->python fast_RSA.py -hd=D

If you wish to setup the center/span: python fast_RSA.py -c=106.6e6 -s=1e6
The center and span units are Hz
