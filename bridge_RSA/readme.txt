Main installation:
	Require a Windows, RSA is a DLL and no version is available for Linux or Mac.
	Install Python 64bit and RSA which are contained on the "toinstall" folder.

--> Relaunch the Python program, go in "modify" --> Press Next --> Check if "Add Python environment variables" is ticked, if not, tick it.
 
Simple launch:
	To launch the script, double click on start_rsa. 
 
	If the shell do not print any "Waiting connection" after 5 seconds, ctrl+C can unlock the issue.

	To stop the script, you will need to ctrl+C and wait few seconds or just close the shell.
